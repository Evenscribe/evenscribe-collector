# Olympus-Observatory

Server implementation for Olympus Observatory.

## Requirements:
- [Cmake](https://cmake.org/)
- [Make](https://www.gnu.org/software/make/)
- [Clickhouse](https://clickhouse.com/)

### Starting Clickhouse Server
```bash
clickhouse server
```

### Building Project
```bash
make dev
```

### Run Executable
```bash
make run
```
